# 💪 Muscle-Mania

Muscle-Mania is a fitness and workout website built using HTML, CSS, and JavaScript.

## Features
- Workout guides for different muscle groups
- Nutrition and diet information
- Calorie calculator
- Exercise directory
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
- index.html
- calories.html
- diet.html
- directory.html
- Muscle group pages
